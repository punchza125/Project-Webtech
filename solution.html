Quick Fix



KIRIN PRO WIRELESS

1. วิธีการดู % แบตเตอรี่เม้าส์
  • - ถ้าแบตเตอรี่ต่ำกว่า10% ไฟจะกระพริบสีแดง
  • - ระหว่างที่ชาร์จ ไฟจะกระพริบสีส้ม
  • - เมื่อแบตเตอรี่เต็ม ไฟจะหยุดกระพริบ แล้วกลับเข้าสู่โหมดที่ตั้งค่าไว้
  • - บนหน้าSoftware จะแจ้งระดับของแบตเตอรี่โดยประมาณ

2. แก้ไขสัญญาณ Wireless เบื้องต้น
  1. ถ้ามีอุปกรณ์สัญญาณ ไวร์เลส อย่าง เร้าเตอร์ อาจจะต้องไว้ให้ห่าง อย่างต่ำ 2 เมตร
  2. ใช้สายต่อออกมา dongle ให้ตัวรับอยู่ไม่เกิน40 cm.
  3. ลองปรับ Polling Rate เป็น500
  4. เปลี่ยนช่อง USB เป็นUSB 2.0

3. วิธีรีเซ็ตสัญญาณเม้าส์
  1. กดเม้าส์ซ้าย ขวา และกลางพร้อมกันจนกว่าไฟจะกระพริบแล้วปล่อย
  2. เมื่อสัญญาณต่อเสร็จไฟจะหยุดกระพริบเอง

4. Software Kirin

5. รีวิว Software Kirin Pro Wireless

A. วิธีการ update ตัวรับสัญญาณ
  1. เสียบตัวรับสัญญาน
  2. เปิดไฟล์ Wireless mouse update for receiver.exe
  3. กด start ที่ปุ่มสีเขียว
  4. รอจนสำเร็จ ห้ามถอดหรือขยับตัวรับสัญญาณแล้วสัญญาณหลุดเด็ดขาด

B. วิธี update firmware เม้าส์
  1. ******ถอดตัวรับสัญญาณไร้สายออกจาก computer *********
  2. เสียบสาย USB C
  3. เปิด Wireless mouse update for mouse.exe
  4. คลิ๊ก start ที่ปุ่มเขียว แล้วรอจน update เสร็จ ห้ามถอดหรือขยับเม้าส์แล้วสัญญาณหลุดเด็ดขาด

RAVANA

1. Software

ลูกค้าต้อง Uninstall Version อื่นๆ ออกก่อน และใช้ได้เฉพาะกับโหมดสายเท่านั้น กด FN+Tab เพื่อสลับโหมดสายและไร้สาย

https://drive.google.com/file/d/15o-oYUeW2abrP-nwuLyQdocLO87Ojc8S/view?usp=sharing

2.การสลับโหมดสาย และไร้สาย

กด FN + Tab เพื่อสลับโหมด ไฟจะกระพริบทั้งคีย์บอร์ด หากคีย์บอร์ดเข้าโหมดไร้สาย ไฟที่ตัว P จะกระพริบให้เห็นแล้วดับไป

3.Update Firmware / P กระพริบ, เข้าหน้า Software ไม่ได้

#อัพเดตfirmware

"Ravana firmware version 513"

- แก้ bug ไม่สามารถบันทึก macro ได้

**สำหรับผู้ใช้ในแบบ plug&play ไม่จำเป็นต้องอัพเดต

#กรุณาอ่านก่อน update นะครับ ---------------------------------------------------

ย้ำอีกครั้งอ่านก่อน update ให้ครบทุกขั้นตอนนะครับ

  1. ถอดตัวรับสัญญาณ
  2. ปิดแบตเตอรี่ ด้านใต้
  3. เสียบสาย USB ให้แน่น เปิดโหมดสาย ด้วย alt + tab
  4. เปิด software Ravana เวอชั่นล่าสุด 1.1.12
  5. กดที่ support ด้านขวากด check update ตรง firmware
  6. กดตกลง แล้วรอจนกว่าจะ 100% ห้ามถอดสายหรือไฟดับหรือปิดคอม
  7. update ครบ 100% ให้ถอดสายเสียบใหม่ 1 ที
  8. update สมบูรณ์ สามารถ save มาโครไปใช้งานได้ทั้งโหมดสายและไร้สาย
